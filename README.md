# Introduction
This is an implementation of a distributed system using .Net Core

# Technologies
* Domain-Driven Design
* Event Sourcing
* CQRS
* Saga Pattern (using [Chronicle](https://github.com/snatch-dev/Chronicle))
* [Kafka](https://kafka.apache.org/)
